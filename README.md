# kOrc.me

Source code for my personal website, https://korc.me

## Local build
```
mkdir korcme
cd korcme
git init
git remote add origin git@github.com:keerthik/keerthik.github.io.git
git fetch
git switch master
git pull
brew install hugo
hugo server -D #to show drafts
```
